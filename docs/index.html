Get Started

Feature 1: User input and user fitness estimation

  • User can click on the calculate fitness button, which takes them to the user profile page

Feature 2 (Sprint 3 Partial Implementation): Show user nearby trails

  • User clicks on the hike tab on top menu bar and is brought to the Hike Page(Link here)
  • User is on the Hike page(Link here) and enters zip code into the search bar
    • The user's location is then stored to search for hikes within a radius
  • User can click on the Calculate Fitness Button and be redirected to the user information page

Feature 4: User can find directions to a trailhead from zipcode

  • User enters the zip code of their current location into the From: form field on each nearby trail.
    • The user can click "Get Directions" and be sent to Google maps, displaying a route to the trailhead
